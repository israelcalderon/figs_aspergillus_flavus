\textbf{{\Large Resumen}}
\vspace{0.9cm}

La contaminación de higos por *Aspergillus flavus* representa un problema crítico para la seguridad alimentaria y la economía agrícola debido a la producción de aflatoxinas cancerígenas. Este trabajo propone un enfoque híbrido y no invasivo para optimizar su detección mediante el uso de imágenes hiperespectrales. Para resolver el desafío computacional provocado por la alta dimensionalidad inherente a estas imágenes, se implementaron dos algoritmos metaheurísticos: Algoritmos Genéticos y Optimización por Enjambre de Partículas. Estas técnicas permitieron identificar subconjuntos óptimos de bandas espectrales clave. La aptitud de cada subconjunto se evaluó utilizando la métrica F1-Score de un modelo de clasificación basado en Máquinas de Vectores de Soporte. Los resultados demostraron que es posible disminuir drásticamente la dimensionalidad de los datos y el costo computacional sin sacrificar la precisión diagnóstica, sentando así las bases para la implementación de futuros sistemas de inspección agrícola en tiempo real.

\vspace{0.9cm}
\textbf{Palabras clave:} Imágenes Hiperespectrales, Metaheurísticas, *Aspergillus flavus*.

\textbf{{\Large Abstract}}
\vspace{0.9cm}

The contamination of figs by *Aspergillus flavus* represents a critical problem for food safety and the agricultural economy due to the production of carcinogenic aflatoxins. This work proposes a hybrid, non-invasive approach to optimize detection using hyperspectral imaging. To solve the computational challenge caused by the high dimensionality inherent in these images, two metaheuristic algorithms were implemented: Genetic Algorithms and Particle Swarm Optimization. These techniques allowed the identification of optimal subsets of key spectral bands. The fitness of each subset was evaluated using the F1-Score metric of a Support Vector Machine classification model. The results demonstrated that it is possible to drastically reduce data dimensionality and computational cost without sacrificing diagnostic accuracy, thereby laying the groundwork for the implementation of future real-time agricultural inspection systems.

\vspace{0.9cm}
\textbf{Keywords:} Hyperspectral Imaging, Metaheuristics, *Aspergillus flavus*.
